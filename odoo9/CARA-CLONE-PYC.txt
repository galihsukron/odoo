Cara clone-compiled odoo9

1. harus ada 2 file : clonepyc.py, py_compile.py
   misalnya, taruh di folder C:\clone
   maka alamat file menjadi "C:\clone\py_compile.py"

2  buka file clonepyc.py, lihat baris paling bawah,
   sesuaikan 3 variabel berikut: "src", "dst", "subs_paths"
	src 		: folder odoo9 yg ada "openerp-server" nya
	dst 		: folder output. Hasil proses akan ditaruh di sini
	subs_path	: mungkin yg ini tak perlu diubah, kecuali multi-addons-paths,

   PERHATIAN: nama folder tidak boleh ada spasi. 
   salah : "C:\My Program"
   benar : "C:\MyProgram"

   PERHATIKAN: value "dst" harus merupakan folder yg belum ada, karena akan dibikin otomatis
   salah: dst = r'C:\Windows' karena folder ini sudah ada
   benar: dst = r'C:\OutputAADC' folder ini harus tidak/belum ada. kalau ada hapus dulu.
   benar: dst =  'C:/OutputAADC' karena char "\" harus diawali dengan string raw "r", atau diganti '/'

3  Setelah disesuaikan, jalankan clonepyc.py.
   lihat hasil nya di direktori output.
